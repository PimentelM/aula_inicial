Operações básicas com dados

Strings ( Textos )

Concatenação ( união de strings )

Operador +
 '{{parteA}}' + '{{parteB}}' : '{{ parteA + parteB}}'

Replace ( substituição de texto )

Método .replace
 '{{replaceString}}'.replace('{{replaceThis}}','{{replaceWith}}') : '{{ replaceString.replace(replaceThis,replaceWith)}}'

Alterar o casing ( Maiusculo ou Minusculo )

Método .toUpperCase // .toLowerCase
 '{{casingString}}'.toLowerCase() : '{{ casingString.toLowerCase()}}' '{{casingString}}'.toUpperCase() : '{{ casingString.toUpperCase()}}'
